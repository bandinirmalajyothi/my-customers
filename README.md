# My Customers

Open-source CRM application for managing customer relationships. Features a Kanban board for pipeline tracking, customer detail management, photo uploads via Firebase Cloud Storage, and real-time data syncing with Firestore. Built with vanilla JavaScript and Tailwind CSS—perfect for learning full-stack web development.
